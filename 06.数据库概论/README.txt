0.小组分工：
	详见 README DETAILS.txt。

1.Definition：
	该文件夹下存有我们数据库的设计信息，包括 E-R 图，关系模式，关系模式表（部分实例）。

	（1）Bicycle_ER.pdf：该 pdf 文件是我们设计的 E-R 图。详见 README DETAILS.txt。

	（2）关系模式：该 pdf 文件使我们设计的初始关系模式。详见 README DETAILS.txt。

	（3）build：该文件中使我们最终使用的关系模式，直接用 SQL 语句进行表达。详见 README DETAILS.txt。

	（4）xlsx：其余四个 excel 文件给出了关系模式表，以数据实例的形式进行展现。详见 README DETAILS.txt。

2.Dataset：
	该文件夹下存有我们的数据，分别是四个表的数据，同样以 SQL 语句进行表达。我们附上我们随机生成数据

	的代码和程序，但是建议最好不要经常去运行它。详见 README DETAILS.txt。

3.make：
	出于我们设计与调试数据库的方便，我们写了一个 make 文件，我们建议使用者使用 make 这个文件，因为它确实

	会给数据库的准备带来方便。使用方法详见 README DETAILS.txt。

4.Code：
	该文件夹中存有我们的程序代码，除去生成 xml 所需要的若干代码外，有 Typedef.h, Main.cpp, Manager.cpp, 

	User.cpp, Mysql.cpp, Function.cpp。详见README DETAILS.txt。

5.Output：
	该文件夹中存有一些输出，包括所要求查询的 SQL 语句、查询结果，XML 输出。
