1.登陆与注册功能：

	若要使用管理员权限登陆，请使用账户名：root；密码：mysql

	若要注册请先输入 create，之后可按提示进行

2.管理员功能：

	管理员可增加、删除、修改、查询用户、车辆、订单、地点的信息，其中：

	增加功能由于参数过于多变且复杂，使用此功能时需要直接输入 MySQL 语句；

	删除功能只可以删除用户和车辆，不可删除订单与地点信息；删除功能会自动先删除订单中相关部分；

	修改功能由于参数过于多变且复杂，使用此功能时需要直接输入 MySQL 语句；

	查询功能可查询用户、车辆、订单、地点的信息，需要提供 ID；

	文档中要求的四个查询放在管理员中，具体操作步骤可进入程序查看。

3.用户功能：

	用户可以查询周围可用车辆的 ID；

	用户可查询自己的历史订单信息；

	用户可修改自己的信息；

	用户可充值、开始行程、结束行程；
